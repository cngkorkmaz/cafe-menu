# Cafe-Menu Project

Bu repoda cafe ve restoranlar için öğün çeşitlerine göre seçim yapabileceğiniz bir yapı oluşturulmuştur. Kategorilerine ayrılmış ürünlere ve fiyat ayrıntılarına  kolay bir şekilde ulaşma imkanı sağlanmıştır. Cafe ve restoranlar için benzer tasarımlar kolaylıkla oluşturulabilir. (In this repo, a structure has been created for cafes and restaurants where you can choose according to meal types. It is possible to easily access products and price details divided into categories. Similar designs can be easily created for cafes and restaurants.)

# Özellikler

 Farklı sistemlerde (bilgisayar, telefon, tablet) sorunsuz şekilde çalışan, tamamen uyumlu responsive yapıda bir tasarımdır. (It is a fully compatible, responsive design that works smoothly on different systems (computer, phone, tablet).)

# Kullanılan Teknolojiler

-HTML, CSS, JavaScript
